Ouvrir, enregistrer et imprimer


Ouvrir un modèle existant

Pour ouvrir un modèle ArchiMate existant qui a été créé dans Archi, choisir l'élément de menu "Ouvrir..." dans le menu principal "Fichier" ou dans la barre d'outils principale. Si le modèle est déjà ouvert dans Archi, cette commande n'aura aucun effet.

Vous pouvez aussi glisser un fichier ".archimate" depuis votre bureau dans la fenêtre de l'arborescence des modèles pour l'ouvrir.

Les versions Windows et Mac d'Archi savent assiocier l'ouverture de l'application depuis le bureau en ouvrant des fichiers ayant l'extension de fichier ".archimate".


Enregistrer un modèle

Un modèle peut être enregistrer sur le disque à l'aide de l'élément de menu "Enregistrer" disponible dans le menu principal "Fichier" ou depuis la barre d'outils principale. Si le modèle n'avait pas été déjà enregistré précédemment, il vous sera demandé un nom de fichier.

La commande de menu "Enregistrer sous..." vous permet d'enrefgistrer le modèle sous un nom différent.


Fermer un modèle dans l'arborescence des modèles

Il est possible d'avoir plus d'un modèle ArchiMate ouverts en même temps dans l'arborescence des modèles. Vous pouvez vouloir fermer l'un de ces modèles de façon à ce qu'il n'apparaisse plus dans l'arborescence des modèles. Ceci est réalisable en cliquant le modèle dans l'arborescence des modèles et en choisissant l'entrée de menu "Fermer le modèle" ou en choisissant l'élément de menu "Fermer le modèle" dans le menu principal "Fichiers". Fermer un modèle ne ke supprime pas et il peut être réouvert dans Archi à tout moment via la commande "Ouvrir". Si le modèle a été modifié, il vous sera demandé de l'enregistrer avant de le fermer.


Imprimer une vue (diagramme)

Une vue (diagramme) peut être imprimée sur une imprimante connectée. Ceci est disponible via la commande de menu "Imprimer..." dans le menu principal "Fichiers". Cette commande de menu n'est accessible que si une vue est ouverte.
