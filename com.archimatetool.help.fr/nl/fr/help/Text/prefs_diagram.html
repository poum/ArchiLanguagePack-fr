Préférence des diagrammes


Général


Préférences générales des diagrammes


Taille de la grille
Définit les espacements dans la grille utilisée dans les vues des diagrammes.

Animation

Affiche les animations pour les annuler/refaire
Définit s'il faut utiliser les animations dans les vues des diagrammes pour certaines des commandes annuler/refaire et quand on change le routeur des connexions de la vue.

Vitesse des animations (ms)
Définit la vitesse des animations en millisecondes.


Vue

La palette est ouverte quand on ouvre les vues
Si elle est activée, la palette dans une vue sera ouverte quand la vue est ouverte.

Afficher les infobulles dans les vues
Si activée, les infibulles dans les vues seront affichées.

Modifier le nom après la création d'un nouvel objet à partir de la palette
Si activée, quand un nouvel élément est ajoutée à l'aide de la palette, vous pouvez immédiatement modifier son nom.


Points de vue

Griser les concepts désactivés dans l'arborescence des modèles
Quand cette option est activée, tous les concepts qui sont désactivés dans une vue sont grisés. Pour plus d'information, voir Points de vues.

Masquer les concepts désactivés dans la palette
Quand cette option est activée, tous les concepts qui sont désactivés dans un point de vue ne sont pas affichés dans la palette. Pour plus d'information, voir Points de vue.

Masquer les concepts désactivés dans le connecteur magique
Quand cette option est activée, tous les concepts qui sont désactivés dans un point de vue ne sont pas affichés dans le connecteur magique. Pour plus d'information, voir points de vue.

Rendre translucides les concepts désactivés dans une vue
Quand cette option est activée, les concepts désactivés dans une vue sont rendus translucides. Pour plus d'information, voir Points de vue.


Apparence


Préférences de l'apparence des diagrammes


Illustrations

Afficher les dégradés dans les illustrations
Si activé, les éléments du diagramme sont dessinés avec un dégradé. Note: les dégradés ne sont pas affichés quand l'image d'un diagramme est exportée au format SVG.

Taille par défaut pour les nouvelles illustrations ArchiMate
Définit la largeur et la hauteur par défaut pour les nouvelles illustrations ArchiMate.

Style d'adaptation des mots pour les illustrations ArchiMate
Définit le style d'adaptation des mots pour les illustrations ArchiMate.

Fort - l'adaptation ne sera appliquée que pour les sauts de lignes valides.
Faible - l'adaptation sera toujours appliquée à la fin des espaces disponible, les césures se faisant au milieu d'un mot.
Troncature - l'adaptation sera toujours appliquée à la fin des espaces disponibles, les césures d'un mot ne se faisant que s'il ne rentre pas.

Croquis

Arrière plan par défaut
Définit l'arrière plan par défaut pour les vues croquis nouvellement créées.


Illustrations par défaut


Préférence des illustrations par défaut des diagrammes


Les illustrations par défaut alternatives sont fournies pour dessiner certaines illustrations dans les vues des diagrammes. Cliquez sur une illustration pour définir la forme par défaut à utiliser lors de la création d'une nouvelle illustration.
