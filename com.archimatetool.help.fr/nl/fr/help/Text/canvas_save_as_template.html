Enregistrer un canevas en tant que modèle

Pour enregistrer un canevas existant en tant que modèle, suivez ces étapes:

  1. Créez un nouveau canevas ou ouvrez un modèle existant contenant un canevas.
  2. Choisissez le canevas dans l'arborescence des modèles, faites un clic droit sur lui et choisissez "Enregistrer le canevas en tant que modèle...". Un assistant va s'ouvrir:



  3. Dans l'assistant, fournissez un nom de fichier pour l'emplacement du fichier modèle, un nom pour le modèle (qui est différent du nom du modèle) et une description.
  4. Choisissez si vous voulez inclure une vignette du canevas dans le modèle.
  5. Cliquez sur "Suivant" pour passer à la page suivante de l'assistant:



  6. Choisissez si vous voulez ajouter le modèle à votre collection. Votre collection de modèles est une liste triée en catégories qui sera affichée dans l'assistant Nouveau canevas à partir d'un modèle". S'il n'y a aucune catégorie de disponible, vous pouvez créer une nouvelle catégorie en cliquant sur le bouton "Nouveau..." dans l'assistant.
  7. Appuyez sur "Terminer".

Le modèle sera enregistré dans votre système de fichiers avec une extension "*.archicanvas". Vous pouvez partager ce modèle avec d'autres utilisateurs d'Archi si vous le souhaitez.
