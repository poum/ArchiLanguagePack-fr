Gestion de vos modèles de canevas

Vous enregistrez vos modèles de canevas dans votre système de fichiers en tant que fichiers "*.archicanvas". Ceux-ci peuvent être enregistrés n'importe où dans votre système de fichiers local. Archi vous permet de créer des collections utilisateurs qui pointent vers ces modèles. Ce sont des raccourcis vers les modèles réels. Pour gérer votre collection de modèles, suivez ces étapes:

  1. Choisissez le dossier "Vues" pour le modèle choisi dans l'arborescence des modèles, faites un clic droit dessus et choisissez "Nouveau->Canevas depuis un modèle...". Un assistant va s'ouvrir:



  2. Choisissez "Gérer..." dans la section "Modèles" sur la gauche de l'assistant. Une fenêtre de dialogue va s'ouvrir:



  3. Ce dialogue vous permet d'ajouter, de renommer et de supprimer de nouvelles catégories de modèles, ainsi que d'ajouter des modèles depuis des fichiers de votre collection. Vous pouvez également modifier le nom et la description de chacun des modèles.
  4. Pour ajouter un modèle depuis un fichier, cliquez sur le bouton "Ajouter un modèle...". Choisissez le fichier "*.archicanvas" dans le dialogue fichier qui apparait.
  5. Pour ajouter une nouvelle catégorie de modèle, cliquez sur le bouton "Nouvelle catégorie...". Donnez un nom pour la catégorie.
  6. Pour ajouter des modèles à une catégorie, glissez-déposez une entrée de modèle de la table "Modèles" dans le dossier catégories dans l'arborescence des "Catégories". Notez qu'un modèle peut apparaître dans plus d'un dossier de catégorie.
