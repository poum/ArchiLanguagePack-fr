Travailler avec Archi

Puisque vous voilà dans l'application, vous pourriez vouloir savoir comment les choses fonctionnent d'une manière générale.

Fenêtres et onglets

La principale zone de modification des vues se trouve au centre à droite de l'application. Les fenêtres et les onglets peuvent être tirés et glissés pour être réarrangez comme vous le souhaitez. Vous pouvez même détacher certaines fenêtres pour les rendre "flottantes".

Si vous souhaitez à un moment donné réinitialiser l'arrangement des onglets et des fenêtres de l'application dans leur état initial, sélectionnez "Réinitialiser l'arrangement des fenêtres" dans le menu principal "Fenêtre". Vous pouvez également réinitialiser l'état de l'application pour restorer les réglages de départ via l'entrée de menu "Restorer les valeurs par défaut" du menu "Aide"'. Les préférences utilisateurs sont préservées dans les deux cas.


Annuler/Refaire (contextuel par modèle)

Des commandes Annuler et Refaire totalement fonctionnelles sont disponibles pour toutes les actions qui sont réalisées par l'utilisateur dans Archi.

Les commandes Annuler et Refaire sont contextuelles et dépendent du modèle sélectionné dans l'arborescence des modèles ou dans une vue. Cliquer sur une vue ou sur l'aborescence des modèles activera la commande si une action a été exécutée pour le modèle en question.


Pense-bêtes

Les pense-bêtes peuvent aider l'utilisateur et le guider via une suite d'étapes de façon à atteindre un certain but. Certaines étapes peuvent être exécutées via le pense-bête et certains sont commentés de telle sorte que l'utilisateur puisse manuellement compléter l'étape. Les pense-bêtes sont accessibles via le menu principal "Aide". Actuellement, Archi comporte deux pense-bêtes - "Créer une vue carte" et "Créer un nouveau modèle".


Obtenir de l'aide

Une aide contextuelle et complète est disponible via le menu principal "Aide". Appuyer sur la touche "F1" dans une fenêtre va faire apparaître l'aide contextuelle.
