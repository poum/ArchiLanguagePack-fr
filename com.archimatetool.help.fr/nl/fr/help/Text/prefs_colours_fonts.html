Couleurs


Préférences des couleurs


Choisir les couleurs par défaut à utiliser pour les éléments et les connexions. Ce sont les couleurs qui seront affichées quand une couleur de remplissage ou de ligne est définie à "Par défaut" dans la fenêtre des propriétés pour un élement ou une connexion du diagramme. Les couleurs par défaut peuvent être définies différemment pour chaqye installation d'Archi, par conséquent, par exemple, un utilisateur peut avoir un ensemble complètement différent de couleurs de remplissage qu'un autre utilisateur. Les couleurs ne seront pas sauvegardées dans le fichier du modèle .archimate à moins que l'option "Enregistrer la couleur de remplissage par défaut pour un nouvel élément dans le fichier .archimate" n'ait été cochée.

Dériver les couleurs des lignes d'un élément à partir des couleurs de remplissage
Si ceci est coché, la couleur des lignes d'un élément du diagramme sera dérivée comme ombre plus sombre de sa couleur de remplissage. Vous pouvez ajuster le contraste avec le contrôle du "Facteur de contraste". Si cette option est cochée, les couleurs des lignes des utilisateurs seront ignorées.

Enregistrer la couleur de remplissage par défaut pour les éléments dans le fichier du modèle
Si ceci est coché, la couleur par défaut d'un élément sera enregistrée dans le fichier du modèle et sera ensuite "figée" dans le fichier. Ceci assure que si ce fichier est partagé avec un autre utilisateur, il verra ces mêmes couleurs.

Utiliser ces couleurs de remplissage dans l'arborescence des modèles et dans la palette de l'éditeur
Si ceci est coché, alors les couleurs de remplissage par défaut utilisée dans l'application dans l'arborescence des modèles et la palette de l'éditeur. Ceci nécessite un redémarrage de l'application pour être pris en compte.
