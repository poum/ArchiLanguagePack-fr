Réglages

Dossier des modèles utilisateur
Définit l'emplacement racine où les modèles utilisateurs des rapports Jasper sont enregistrés. Chaque modèle doit être dans son propre sous-répertoire. Quand on exporte un modèle au format de rapport Jasper, le nom du répertoire doit être affiché dans l'assistant d'export. Pour plus d'informations, voir Génération de rapports.
